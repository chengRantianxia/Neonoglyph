# 程序员的小盒子

一个简单的书签管理工具，专为程序员设计。

## 特性

- 📚 分类管理：支持创建多级分类，方便整理
- 🏷️ 标签系统：灵活的标签管理，快速查找
- 🌓 暗黑模式：护眼模式，提供舒适的阅读体验
- 🌍 国际化：支持中英文切换
- 💾 本地存储：数据存储在本地，保护隐私
- 📤 导入导出：支持数据备份和迁移
- 🔍 快速搜索：实时搜索，快速定位

## 技术栈

- Vue 3
- TypeScript
- Vite
- Vue Router
- Pinia
- Element Plus
- VueUse
- Dexie.js

## 开发

```bash
npm install
npm run dev
```
